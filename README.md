# Record daily inmate data

Saves [Daily Inmates In Custody](https://data.cityofnewyork.us/Public-Safety/Daily-Inmates-In-Custody/7479-ugqb/about_data) data. 

Automatically runs once per day (with [Daily Data Fetch](.github/workflows/daily_data.yml) action).

Saves raw data in [Daily data](./Daily%20data/).

Saves summary data in [Summary data](./Summary%20data/)
